# Day 09: System Information in Linux

## Overview
System information commands are used to get details about the Linux system,
such as kernel version, disk usage, and system name. These commands help
understand the system configuration and storage status.

---

## Commands

### uname
**Purpose:**  
Used to display system and kernel information.

**Syntax:**
- uname
- uname -a

**Usage:**
- Shows kernel name and version
- Useful for system identification

---

### df
**Purpose:**  
Used to check disk space usage.

**Syntax:**
- df
- df -h

**Usage:**
- Shows available and used disk space
- `-h` displays output in human-readable format

---

### du
**Purpose:**  
Used to check disk usage of files and directories.

**Syntax:**
- du
- du -h

**Usage:**
- Helps find large files or folders
- Useful for storage cleanup

---

### hostname
**Purpose:**  
Used to display the system hostname.

**Syntax:**
- hostname

**Usage:**
- Identifies the system on a network

---

## Summary
- uname → system and kernel details  
- df → disk space usage  
- du → directory size  
- hostname → system name
