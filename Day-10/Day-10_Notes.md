# Day 10: Networking Information in Linux

## Overview
Networking commands are used to check network connectivity,
IP configuration, and network status. These commands are essential
for troubleshooting network-related issues.

---

## Commands

### ping
**Purpose:**  
Used to check network connectivity.

**Syntax:**
- ping hostname

**Usage:**
- Checks if a system is reachable
- Tests internet connection

---

### ifconfig
**Purpose:**  
Used to display network interface details.

**Syntax:**
- ifconfig

**Usage:**
- Shows IP address and network configuration

---

### netstat
**Purpose:**  
Used to display network connections.

**Syntax:**
- netstat

**Usage:**
- Shows active network connections
- Useful for network troubleshooting

---

### ss
**Purpose:**  
Used to display socket statistics.

**Syntax:**
- ss

**Usage:**
- Faster alternative to netstat
- Shows active connections

---

## Summary
- ping → network connectivity  
- ifconfig → interface details  
- netstat → network connections  
- ss → socket statistics
