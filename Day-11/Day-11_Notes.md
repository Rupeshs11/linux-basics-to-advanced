# Day 11: Accessing Remote Server in Linux

## Overview
Linux allows users to access remote servers securely using command-line tools.
These commands are widely used in cloud computing and server administration.

---

## Commands

### ssh
**Purpose:**  
Used to connect to a remote server securely.

**Syntax:**
- ssh username@server_ip

**Usage:**
- Secure remote login
- Encrypted communication

---

### scp
**Purpose:**  
Used to copy files between local and remote systems.

**Syntax:**
- scp file username@server_ip:/path

**Usage:**
- Secure file transfer
- Commonly used with SSH

---

## Summary
- ssh → connect to remote server  
- scp → copy files securely
